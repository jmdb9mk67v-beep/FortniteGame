How To Compile And Run;

1.  Open your terminal and make sure you are in the root project folder.
2.  To Compile all the necessary classes at once, run the following command:

    javac audio/*.java combatants/*.java items/*.java util/*.java logic/*.java

3.  To launch the game, run:

    java logic.Game
